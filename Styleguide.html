Code Styleguide

Ein Styleguide hilft dabei, den Code konsistent und leserlich zu halten. Hier sind einige wichtige Richtlinien:

1. Einrückung

Verwende 2 Leerzeichen für die Einrückung.

function example() {
  console.log("Hello, world!");
}

2. Variablen

Nutze const für unveränderliche Werte und let für veränderliche. Vermeide var.

3. Namen

Verwende camelCase für Variablen und Funktionen.

let userName = "Amine";
            or var variable = 2;

4. Kommentare

Nutze // für Einzelzeilenkommentare und /* */ für Blockkommentare.

// Dies ist ein Einzelzeilenkommentar
/* Dies ist ein Blockkommentar */

5. Semikolons

Setze immer Semikolons ans Ende einer Anweisung > ";"

6. Funktionen

Bevorzuge Pfeilfunktionen, wenn sie lesbarer sind.

const add = (a, b) => a + b;
